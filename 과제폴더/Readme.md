# 1.게임컨셉
###### 기본적으로 가로방향으로 무한 스크롤링되는 맵을 중력의 영향을 받는 새를 조종하여 장애물에 맞지 않고 점수를 올리는 게임 입니다.  
  
# 2.게임 개발 범위

#### 2-2. 중력의 영향을 받는 환경  

#### 2-3. 맵상에 랜덤으로 생성되는 장애물  

#### 2-4. JSON을 이용한 최고점수 저장 및 불러오기
